Target audience:¶

Advanced undergraduate students in Chemistry. This course should also be of interest for students in Applied Mathematics, Physics, Chemical Engineering, Pharmaceutical Science, and related, interested in computational modeling, molecular visualization and data analysis. In addition to students working in computational modeling, the course is useful for anyone interested in practical aspects of modern simulations. The data analysis and visualization parts are also applicable to analysis / visualization of experimental data.

Pre-requisites:¶

Chemistry 2272F, 2281G, 2283G, 2384B. Unless you have either the pre-requisites for this course or written special permission from your Dean to enroll in it, you may be removed from this course and it will be deleted from your record. This decision may not be appealed. You will receive no adjustment to your fees in the event that you are dropped from the course for failing to have the necessary prerequisites.

Practical requirements¶

  1. Installation of Gromacs (version 2018.6) is required. This means: either the ability to install it on your own personal laptop or having access to Gromacs otherwise (that access must be available during labs). For installation on your own computer: Works easily if your operating system is either Windows 10, Linux, or OSX. Detailed instructions will be provided during the first lecture / labs. About Gromacs: Gromacs is one of the world’s most popular and fastest molecular simulation software and available freely (open source) at www.gromacs.org

  2. VMD. VMD (Visual Molecular Dynamics) is a powerful tool for analysis and visualization of molecules. It is freely available at http://www.ks.uiuc.edu/Research/vmd/. VMD is available for all common operating systems (Linux, Windows, OSX).

  3. Python installation. Some python programming is included in the course. Python is available for all common operating systems at www.python.org

  4. Plotting program like grace, gnuplot, Matlab, etc.

Software that needs to be installed on your computer during this course¶

The list below provides a rough idea as what software you need to install during this course. We will work through all of them in detail when they are needed.

  • Linux (unless you have a Mac)

  • Python

  • Jupyter Lab/Notebook

  • Gromacs version xx.xx. Gromacs is a state-of-the-art simulation package and it will be used throughout the course.

    • Gromacs has some requirements including a c/c++ compiler, cmake and so on. We will discuss all these details when we discuss Gromacss.

  • VMD - Visual Molecular Dynamics. It is freely available at http://www.ks.uiuc.edu/Research/vmd/ for all common operating systems (Linux, Windows, OSX).

  • Plotting program. Jupyter Lab/Notebook is the primary environment for data analysis and visualization but it is good to have other options such as grace or gnuplot.

IMPORTANT: You are solely responsible for virus checking and security for all your software installations.

What kind of a computer / resources / operating system do I need for this course?¶

For the purposes of this course pretty much any reasonable modern computer is fine. Say, if your computer is able to run Windows 10, it will work. What will be needed, however, is an installation of Linux. If you are a Mac users, you don’t need necessarily to do that since you already have a CLI but you need to pay some extra attention to some of the commands since there are some differences beween Linux and Mac terminals. In particular, Linux uses the bash shell and Mac uses zsh.

Notes:

  • We are not running massive simulations so you should be able to perform the tasks on any reasonable modern computer

  • You will need to install Linux (three possible ways to do that will be discussed in detail below) unless you have a Mac.

  • If you have a discrete NVIDIA GPU, you will be able to use it for running simulations provided you have either pure Linux or dual boot. This will be discussed when we discuss Linux and software installation.

  • You will need some storage space. 10 GB should be more than enough.

Topics to be covered¶

The aim of this course is to familiarize the student with modern methods and methodology in computational chemistry. Emphasis is put on direct hands-on experience using production-quality software and methods.

  • Introduction to Linux/Unix operating system

    • Linux is the dominant operating system in all High Performance Computing (HPC). Since 2017, all of the world’s TOP500 computing facilities have been using Linux.

    • Basic of the command shell (terminal)

    • Basic installation and maintenance of software

    • Security issues

  • Data management and workflow

  • Basics of python programming

  • Introduction to Machine Learning

  • Introduction to ab initio (quantum level) modeling

  • Classical molecular dynamics

  • Visualization and plotting

molecular dynamics software (see below). The course provides a broad overview of the common methods in the framework of molecular modeling and, importantly, how to relate the data to experiments. Simulations of proteins, lipid bilayers and solvation of organic molecules will be performed. The course will be a balanced treatment of modeling, data analysis, visualization and relation to experimentally observable / measurable phenomena.
